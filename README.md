🪄 Magic Wand (POV Display) - PCB Design
This repository contains the hardware design files for the Magic Wand project, inspired by Andre LaMothe's "Crash Course Electronics and PCB Design".

The Magic Wand is a classic Persistence of Vision (POV) device. When the PCB is waved through the air, the precisely timed LED strobing creates the illusion of floating text or graphics.

🛠 Project Structure
The repository is organized to provide everything needed for manufacturing and further development:

Magic Wand.kicad_sch: The schematic diagram showing the circuit architecture.

Magic Wand.kicad_pcb: The physical board layout, optimized for a handheld wand form factor.

Magic Wand.kicad_pro: The KiCad project management file.

Manufacturing/: This folder contains the necessary output files (such as Gerbers) for PCB fabrication.

🚀 Technical Details
Software: Designed using KiCad.

Concept: Utilizes the POV (Persistence of Vision) effect.

Design Focus: Compact handheld design, signal integrity for high-speed LED switching, and ease of assembly.

💡 Usage
To use these files:

Open the .kicad_pro file in KiCad to view or modify the schematics and PCB layout.

Use the files in the Manufacturing/ directory to order the board from any PCB manufacturer.
